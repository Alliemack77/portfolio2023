Interactive Comments Section: A Frontend Mentor Challenge

This project is a solution to the Interactive Comments Section challenge on Frontend Mentor. This is a fully responsive component built with React and Sass. Users can post, edit, up/downvote, reply to and delete comments. I learned how to build a custom hook that allows for rendering elements based on the viewport size. I also learned accessability best practices for buttons without text, making this commponent keyboard navigation and screen reader friendly.

The View Transition API: Smooth Transitions for Better UX

The View Transition API allows developers to easily create custom transitions between different states and pages. Smooth transitions can reduce the effort needed to navigate, can help users understand where they are within your site, and can making pages feel faster and more responsive. In this video tutorial I demonstrate how to leverage the crossfade provided by the API, and even show how to customize the transitions using CSS animations.
